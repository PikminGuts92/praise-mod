## GP Tool
Multi-purpose tool for creating Guitar Praise customs

### Convert CH songs to GP
`gp_tool packcreate -i 5 -n "Song Pack I" ./ch_songs/song_pack_1 ./gp_songs`

Limitations
- Only audio encoded as ogg vorbis is supported
